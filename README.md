 

Hamiltonian Calculation (upper_base , lower_base , H_ground )



## Physics
Solves the time-independent Schrödinger equation using finite difference method.

## Features
- 1D potential barrier
- Wavefunction simulation
- Transmission and reflection calculation
- Visualization of results

## Tools
- Python
- NumPy
- SciPy
- Matplotlib

## Results
Includes:
- Wavefunction plots
- Transmission vs energy graph
